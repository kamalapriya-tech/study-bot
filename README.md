# StudyBot - AI Chatbot

## Features
- AI-powered chatbot using Groq API
- Real-time responses
- Typing animation UI
- Clean chat interface

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express
- Groq API

## How to Run
1. Clone repo
2. Run `npm install`
3. Add `.env` file:
   GROQ_API_KEY=your_key
4. Run `node server.js`
5. Open index.html
